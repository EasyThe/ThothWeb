Commands

Usage Description Alias
stats Player Name Display stats for the provided PlayerName. stat, pc, st, stata, ст, статс, ns
gods Overall information about the gods in the game and current free god rotation.
god God Name Provides information about God Name. g
rgod Gives you a random God and randomised build. rg, randomgod, random
rteam number Gives you number random Gods with randomised builds for them. team, ртеам, теам, rt
status Checks the status page for the status of Smite servers. статус, statis, s, с, server, servers, se, се, serverstatus
statusupdates #channel When SMITE incidents and scheduled maintenances appear in the status page they will be sent to #channel statusupd, su
stopstatusupdates Stops sending messages from the SMITE status page. ssu
item Item Name Provides information about Item Name. i
trello Checks the SMITE Community Issues Trello Board. issues, bugs, board
livematch Player Name or MatchID Match details if provided Player Name is in a match. live, lm, l
matchdetails Player Name or MatchID Match details for the provided MatchID> or latest match played of provided Player Name. md, мд
matchhistory Player Name Latest match history for Player Name. mh, мх, history
motd Information about upcoming MOTDs in the game. motds, мотд, мотдс
wp Player Name Shows all masteries for gods played by the provided Player Name. wps, worshipers, mastery, masteries
link Provides information about the linking process.
startlink Starts the linking process for your Discord and SMITE accounts.
unlink Unlink your SMITE and Discord accounts in Thoth's database.
rank Gives you random ranked division. ранк
feedback Feedback Message If you have got feedback for the bot, this is the command.

About Thoth

Thoth is a Discord bot for SMITE player stats, match details, live match details, gods, SMITE server status, random gods, random builds and more!
